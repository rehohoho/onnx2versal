Detailed Description

  • qy = saturate ((y / y_scale) + y_zero_point) => x = (qx - qx_zero) * qx_scale
  • Softmax(input, axis) = Exp(input) / ReduceSum(Exp(input), axis=axis, keepdims=1)

For exp(x) ~= (1 + x/256)**256, Let x1 = 1+x/256 = 1 + (qx-qx_zero)*qx_scale/256 = qx*qx_scale/256 - qx_zero*qx_scale/256 + 1 Then x' = x1*x1 (8 times)

For y = exp(x)/div, qy = exp(x)/div/qy_scale + qy_zero
