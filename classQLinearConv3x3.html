QLinearConv3x3<28,32,24,32,1,1,6,5>

int16 * int8: expands 9 weights into 16 long vector [a,b,c,0, d,e,f,0, g,h,i,0, 0,0,0,0]

stride1 acc0 += z0*x0 + z1*x1 z2*x2 + z3*x3 acc1 += z0*x1 + z1*x2 z2*x3 + z3*x4 ... acc14 += z0*x14 + z1*x15 z2*x16 + z3*x17 acc15 += z0*x15 + z1*x16 z2*x17 + z3*x18

stride2 acc0 += z0*x0 + z1*x1 z2*x2 + z3*x3 acc1 += z0*x2 + z1*x3 z2*x4 + z5*x4 ... acc6 += z0*x12 + z1*x13 z2*x14 + z3*x15 acc7 += z0*x14 + z1*x15 z2*x16 + z3*x16

Vector registers can hold 256 int8 at most, 128 int16 at most.

xoffsets: 4b offset for lane 0,2,4,6, for 04, off0=2*4, off2=(0+4 +1)*2 => 8,9, 10,11 xoffsetshi: 4b offset for lane 8,10,12,14, same selection scheme
